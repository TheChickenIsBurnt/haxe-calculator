# haxe-calculator
Calculator in Haxe :D

# how to compile:

you will need Haxe installed to compile this.
then go to the directory you installed it into 
and type in:

```haxe -main Main --interp```

then it will be compiled!!1! XDDD

# OR

you can just type in a shell prompt:

```chmod +x run.sh```
and
```./run.sh```
